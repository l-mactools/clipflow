# ClipFlow

一个原生、轻量、本地优先的 macOS 剪贴板历史工具。

## 当前能力

- 监听文本、链接、文件和图片类型
- 本地历史记录、去重、搜索和分类筛选
- 收藏、删除及一键清理未收藏记录
- 对常见 Token、密钥和密码格式进行过滤
- 菜单栏入口与原生 SwiftUI 三栏界面
- 可自定义全局面板快捷键
- 最近五条记录可分别配置全局快捷键，无需打开面板即可复制

## 运行

要求 macOS 14 与 Xcode 15.3 或更高版本。

```bash
swift run
```

也可以在 Xcode 中打开 `Package.swift` 后运行 `ClipFlow` scheme。

## 隐私

历史记录仅写入本机 `Application Support/ClipFlow/history.json`。当前版本没有网络请求和云同步。

## 默认快捷键

- `⇧⌘V`：打开 ClipFlow
- `⌥⌘1` 至 `⌥⌘5`：直接复制最近第 1 至第 5 条记录

可从主窗口工具栏或菜单栏的“快捷键设置…”重新录制。
