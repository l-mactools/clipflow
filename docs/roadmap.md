# 拾笺路线图

拾笺的方向不是做一个更复杂的剪贴板仓库，而是做复制后的工作流入口。核心体验始终保持：快捷键呼出、键盘选择、立即取回。

## v0.2 基础体验

目标：让快捷面板足够顺手，成为日常默认入口。

- 面板内直接输入搜索，无需切到完整窗口。
- `⌘1` 到 `⌘9` 快速选择最近记录。
- `Enter` 复制，`Command + Enter` 复制并粘贴回当前应用。
- 自动去重、限制历史容量、默认 24 小时后清理过期普通记录。
- 支持统一历史有效期，也支持按文本、链接、图片、文件分别设置保存时长。
- 更可读的链接标题；默认不根据内容主动拦截本地剪贴板。

## v0.3 复制栈与临时篮子

目标：从“找回某一条”升级为“组织刚复制的一组内容”。

- 复制栈：按顺序取回最近多条内容，支持连续粘贴。
- 临时篮子：把多条历史加入篮子，重排后一次性取用。
- 多条合并：输出为纯文本、Markdown 列表、引用块或 Prompt 上下文。
- 篮子内容支持固定、清空和从快捷面板快速加入。

## v0.4 上下文感知

目标：让历史按当前工作场景自动变得相关。

- 记录来源 App、窗口标题和来源时间段。
- 快捷面板优先展示当前 App 相关内容。
- 支持按 App、类型、时间段过滤。
- 对浏览器链接、文件路径和开发项目提供回到来源的入口。

## v0.5 内容动作

目标：把复制内容变成可直接处理的对象。

- URL 一键转 Markdown 链接。
- JSON 格式化、压缩和校验。
- 文本去换行、去空格、大小写转换。
- 代码片段包成 Markdown 代码块。
- 图片保存到桌面、复制为 PNG、查看尺寸和文件大小。

## v1.0 产品化

目标：达到长期使用和公开分发的稳定标准。

- 完整偏好设置：快捷键、历史容量、统一保留时长、按类型保留时长、忽略 App。
- 启动项、自动更新、崩溃恢复。
- Apple 签名与公证，完善 DMG 安装体验。
- 本地数据管理和历史清理工具。
